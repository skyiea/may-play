# jsx-suit
Quick-start suit for JSX development

Includes:
  - Webpack (with hot dev-server)
  - React
  - ES6 and JSX compiler (babel)
  - LESS compiler
  - Nodemon server
  
To prepare project (install necessary global and local packages), use:
  - npm prepare
  
To start suit:
  - npm run server
  - npm run hot
  - open http://localhost:1234
